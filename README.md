# 📚 학생 과제 관리 시스템

## 프로젝트 소개
학생들의 과제 마감일을 효율적으로 관리하고, 마감이 임박한 과제를 한눈에 확인할 수 있도록 도와주는 웹 애플리케이션입니다.  
단순한 일정 나열이 아니라, 마감일까지 남은 시간(D-day)을 자동으로 계산하고 과제 진행 상태를 시각적으로 구분하여 사용자가 현재 가장 중요한 과제를 한눈에 파악할 수 있도록 설계되었습니다.

특히 고등학생의 학습 환경을 고려해
✔ 복잡한 기능은 줄이고
✔ 꼭 필요한 정보만 빠르게 확인할 수 있도록
직관적이고 시원한 UI를 목표로 개발하였습니다.

## 개발 기간
2025.12.19

## 주요 기능
1️⃣ 과제 등록 및 마감일 설정
과제명, 과목명, 마감일을 간단한 입력으로 등록
여러 과제를 동시에 관리 가능
수정 및 삭제 기능 제공

2️⃣ D-day 자동 계산
마감일까지 남은 기간을 D-day 형태로 자동 계산
마감이 가까워질수록 사용자가 쉽게 인지할 수 있도록 표시
날짜 계산을 통해 긴급한 과제 우선 파악 가능

3️⃣ 과제 완료 / 미완료 상태 관리
체크 한 번으로 과제 완료 처리
완료된 과제와 미완료 과제를 시각적으로 구분
완료 여부에 따라 화면에서 자연스럽게 정리되어 가독성 향상

## 기대 효과

과제 마감일을 놓치는 실수를 줄일 수 있음
현재 해야 할 과제를 명확히 인식 가능
학습 일정 관리 습관 형성에 도움
고등학생도 부담 없이 사용할 수 있는 간단한 과제 관리 도구 제공
