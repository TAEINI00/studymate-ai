# 📚 학생 과제 관리 시스템

## 프로젝트 소개
고등학생들의 과제 마감일을 효율적으로 관리하고, 마감이 임박한 과제를 한눈에 확인할 수 있도록 도와주는 웹 애플리케이션입니다.  
과제 등록부터 D-day 계산, 완료 여부 관리까지 간단하고 직관적으로 사용할 수 있도록 설계되었습니다.

## 개발 기간
2025.12.19

## 주요 기능
- 과제 등록 및 마감일 설정
- 마감일까지 남은 기간(D-day) 자동 계산
- 과제 완료 / 미완료 상태 관리

## 사용 기술
- Frontend: HTML, CSS, JavaScript  
- Backend: Python Flask  
- Database: SQLite
